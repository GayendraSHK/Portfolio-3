
# My Portfolio Website

This portfolio website showcases my skills, projects, and experience in software engineering and design. It's designed for potential employers and collaborators to easily explore my work and connect.
## Run Locally

Clone the project

```bash
  git clone https://github.com/GayendraSHK/My-Portfolio.git
```

Go to the project directory

```bash
  cd My-Portfolio
```

Install dependencies

```bash
  npm install
  # or
  yarn install
```

Start the server

```bash
  npm run dev
  # or
  yarn dev
```
